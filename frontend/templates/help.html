Instructions

To upload an image for classification:

  1. Click on the "Get Started" button on the main page.
  2. You will be redirected to the upload page where you can select an image file.
  3. Ensure the image is in a supported format (e.g., JPG, PNG).
  4. Click the "Upload" button to submit your image.
  5. The application will process the image and display the classification result.

If you have any issues, please contact support.
